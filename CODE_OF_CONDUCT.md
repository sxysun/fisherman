# Code of Conduct

This project expects direct, respectful collaboration.

Do:

- keep discussion focused on the technical issue
- assume privacy and security concerns are legitimate until checked
- give concrete reproduction steps and evidence
- respect that contributors may decline unsafe debugging requests involving secrets or private context

Do not:

- post private keys, tokens, real context archives, screenshots, or transcripts
- harass, threaten, or target contributors
- pressure maintainers to bypass privacy safeguards without a clear development-only reason

Reports can be sent privately to the repository owner.
